Run the Project Under Poker\Pokerhands\Pokerhnads.sln

This solution file has the API and the react project.

The solution is configured to start the API and the Project together

If there is an issue with CORS 
install extension https://chrome.google.com/webstore/detail/allow-cors-access-control/lhobafahddgcelffkeicbaginigeejlf

There are two inputs for the Project - Player 1 and Player 2 Name. If no name is passed, Player1 and Player2 are defaulted.
